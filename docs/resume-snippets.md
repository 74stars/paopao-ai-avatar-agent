# 简历项目描述

## 项目名称

Paopao 泡泡｜Coze AI 数字分身 Agent

## 一句话版本

设计并实现一个面向个人长期成长的 AI 数字分身 Agent，基于飞书/Coze 工作流接收多模态生活输入，沉淀长期记忆与自我模型，并通过 RAG 和角色化 Prompt 生成主动提醒、目标牵引与每日复盘。

## 简历项目经历版本

**Paopao 泡泡｜Coze AI 数字分身 Agent｜产品设计 & Agent 原型**

- 从 AI 分身社交互动场景出发，设计一个面向个人长期成长的数字分身 Agent，覆盖用户输入捕捉、长期记忆、RAG 检索、自我模型、主动提醒和 Web 展示原型。
- 梳理 AI 分身产品架构，将用户输入拆分为原始档案、结构化记忆、向量记忆和 Self Model 四层，支持目标、欲望、阅读、人物、地点、日程等多维度沉淀。
- 设计 Coze Workflow 与角色化 Prompt，包括输入分类、记忆抽取、RAG 检索、自我模型更新、正向回复生成和早晚主动提醒。
- 搭建飞书机器人 MVP 骨架，支持消息事件接收、原始日志写入、自我模型 JSON 更新以及 `/goal`、`/book`、`/ideal`、`/profile` 等命令。
- 基于用户反馈快速迭代产品定位：从人格标签式日记工具，调整为正向、行动导向的“七年跃迁数字超我”，避免 AI 分身被单一人格标签限制。

## 面试 30 秒介绍

泡泡是我做的一个 AI 数字分身 Agent 项目。它不是普通日记，也不是单纯情绪陪伴，而是一个长期记忆系统：用户可以把想法、欲望、计划、链接、阅读、地点和日程发给它，它会写入原始档案、结构化记忆和自我模型，再通过 RAG 和角色化 Prompt 生成更懂用户的回复、提醒和复盘。这个项目里我重点做了产品定位、Coze Workflow、Prompt 设计、飞书机器人 MVP 和 Web 原型，核心思考是 AI 分身不应该只是人设聊天，而应该逐步连接用户真实生活，成为一个能主动推动用户行动的长期伙伴。
