Product Thinking

不是人设聊天

泡泡探索的是 AI 分身如何从角色互动走向长期记忆、真实生活连接和主动成长牵引。

长期记忆系统

设计 Raw Archive、Structured Memory、Vector Memory 与 Self Model，支持 RAG 检索与画像迭代。

主动正向牵引

不把用户困在负面标签里，而是将欲望、思考和偏离转化为计划、认知资产和下一步行动。

MVP Scope

  • 飞书机器人骨架：事件接收、日志写入、自我模型更新。
  • Coze 工作流设计：输入分类、记忆抽取、RAG、自我模型、回复生成。
  • Prompt 体系：数字超我、正向显化、早晚提醒、日记复盘。
  • Web 原型：捕捉、日记、卡片、日程和蓝色温馨视觉方向。
